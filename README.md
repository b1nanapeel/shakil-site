# shakilmusthafa.com

Personal site for Shakil Musthafa. Built with Astro, plain CSS, zero client-side JS.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Editing content

All content lives in typed data files:

- `src/data/site.ts` for name, bio, and links
- `src/data/work.ts` for the Notable Work section
